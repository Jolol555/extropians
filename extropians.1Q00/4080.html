Before throwing out your vitamin C pills, you should read about the
controversy two years ago regarding vitamin C and genetic damage, "Vitamin C
Report Stirs Controversy," at:
http://www.lef.org/magazine/mag98/jun98_hea.html

Also, see also the material cited in "How Vitamin C Prevents Heart
Attacks"at: http://www.lef.org/magazine/mag99/may99-report3.html

I await the publication (on the web or elsewhere where I can access it) of
the actual study.

One question I would have, if vitamin C is so horrendously bad, as the
report makes it sound, then shouldn't there be a huge positive correlation
between vitamin C supplementation and heart disease? We have lots of people
out there taking more than 500 mg per day, yet we actually see the opposite.
(Per the Life Extension Foundation, vitamin C should be taken with vitamin
E.) I know, cofactors!:) It will be interesting to see what the actual
study says and what was used in it.

(Interesting note: I first read about this study yesterday morning (EST)
when a friend passed me along a URL --
http://www.theage.com.au/breaking/0003/03/A52216-2000Mar3.shtml -- from an
Australian news source.:)
