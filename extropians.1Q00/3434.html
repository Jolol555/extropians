When you subscribe to a mailing list, the first message you receive is
usually a message outlining the procedure for unsubscribing. It is wise
to save that message for when you wish to unsubscribe.

Here it is:

*********************************************************
Welcome to the extropians mailing list!

Please save this message for future reference. Thank you.

If you ever want to remove yourself from this mailing list,
you can send mail to <majordomo@extropy.org> with the following
command in the body of your email message:

    unsubscribe extropians

*********************************************************
