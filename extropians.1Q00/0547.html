On Sun, 9 Jan 2000, Harvey Newstrom wrote:

> Are these "exi" lists owned by the same "ExI" that Max More founded? Why is
> it running from e-groups instead of from the ExI servers that runs this
> list? Where can I get a list of such "exi" lists?

I doubt it. I believe anyone can create "egroups". The list is probably
there because there is no "relative" centralized control. The ExI list
doesn't have much of that either (sometimes to our detriment since we
have to motivate ourselves to improve things). My suspicion would be
that Mike and/or others may have created the group in response to an
effort in mid-'99 to implement a "cooling off" period following some
rather intense discussions about guns that devolved into virtual rock
throwing and name calling.

I believe anyone can query "egroups.com" for "public" groups, you would
have to know it exists and get its name, if it is a "private" group.
