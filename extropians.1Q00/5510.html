The last couple of postings I have made to extropians, I received a
message like:

: Your posting (Subject: "Re: Waco Test Shows "OBVIOUS GUNFIRE"") has
: been sent to the moderator ("nanotechnology-owner@egroups.com") for
: approval. When the moderator approves the posting, it will be
: automatically distributed to the group.

It appears to me that someone has subscribed a moderated egroups
mailing list to extropians. When we post to extropians, the messages
get delivered to nanotechnology@egroups.com, and since it is a moderated
list the egroups server sends this warning to the poster.

I request that something be done about this; either the egroups
nanotechnology server should stop sending these messages, or else that
address should be unsubscribed from extropians.

Hal
