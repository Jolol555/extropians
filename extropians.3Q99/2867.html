Giving new cards means giving cards none of which are the same as any card in the previous batch. Therefore if you are looking in the new batch for a card from the previous batch, you won't find it. It will be missing, because it has been taken away. The trick is that all the cards in the old batch have been taken away, so no matter which card you picked, it will be missing from the second batch.
