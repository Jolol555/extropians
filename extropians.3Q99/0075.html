I have heard about a keyboard that could be designed that would be one step beyond the Dvorak. It would have just one row of keys, and key combinations would signify the letters. So keys 1 and 5 together could represent "B", keys 3, 7 and 8 could represent @, and so forth. The name of this idea is a "chord board".

So far as I know, one has never been produced, and a web search turned up MIDI stuff. Maybe I have the wrong name. But I was thinking this would make a great interface for a wearable keyboard - a pressure sensitive glove would allow you to type while playing "air piano".

Any thoughts?
