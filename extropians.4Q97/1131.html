Mitchell Porter writes:
> I would like there to be a transhumanist discussion list which
> focuses on topics like:

Interestingly enough, I had the same thought. some time ago I created
a list on my box to cover similar issues, and never got around to
announcing it. Here at work we use the term Headspace to describe
mental capacity. When a user does something particulary dumb, we refer
to it as a Insufficient User Headspace Error. So I called the list
'headspace'.

To subscribe, send an email with just the word subscribe in the
subject to headspace-request@x4.net.

Submissions will go to headspace@x4.net.

The original impetus for the list was to discuss ways to enhance
mental performance. I personally would like to know what people out
there know or suggest about Attention Deficit Disorder in adults.

I'm open to suggestions on a charter, or lack thereof. Someone should
at least write an info file to be sent out at subscribe.

I can also provide web space for a web page dealing with these issues,
or any >H issues, really.

-Chris Thompson
