A new mailing list for the >Web/Web++ project has been started and is looking for contributors.

Nextweb is intended for the discussion, design, and eventually implementation of the next generation World Wide Web. Topics include (but are not restricted to) semantically enhanced documents, intelligent documents, agent-based clients, and collaborative filtering.
