There is a discussion of something similar in rec.arts.sf.science right now.

One interesting idea that came out from the discussion is a way of
quantifying tech levels: a tool you can just pick up is tech level 0,
something you can build using level 0 tools is level 1 (flint knife),
something that can be made with level 0 and 1 tools level 2 and so
on. It would be interesting to see how many levels up we have reached
(my guess not that many, the real issue is the breadth of the tech
tree).
