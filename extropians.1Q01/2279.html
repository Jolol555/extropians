If you liked Babylon 5, you'll like Andromeda. A bit of a slide
backwards towards contrived tech like Star Trek, but given the original
author, this should be no surprise. Frankly, I wish Mr. Roddenberry

would have been like it is now - though, of course, one can never know
if the aspects one likes would have been seen as positive back then.
