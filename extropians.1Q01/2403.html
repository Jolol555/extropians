Emlyn wrote:
> We're all dying.

If you say so. I just noticed how much less I felt like that once I
stopped feeling stupid because M$ products seemed bloated and
incomprehensible, as did other things such as the unfolding Java and
Ecmascript product(s) pushed by other players. I figured out that it
might not all be my fault, that I might not be the stupid one, and that
a product with forty million features, some that work, is not something
I need to buy.

The last straw for me was when M$ implemented a bunch of book-publishing
features in a version of Word--just long enough to FUD Pagemaker--then
dropped all those features in the next version of Word. I got the
message loud and clear. So I hung up the phone. I use Win98, but that's
it.

We might all be dying, but some of us are CRAN-conscious.
