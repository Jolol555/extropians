I know a lot of people here use Folding@Home, and we're all Google junkies,
so....
I got this from the infojunkies community on LiveJournal. original URL is
http://www.livejournal.com/talkread.bml?journal=infojunkies&itemid=145998

If you have a windows computer with the Google Toolbar installed, you can
allow it to participate in the Folding@Home project.

Do this:
1. Make sure you have version 1.1 of the toolbar.
2. open regedit (Start->Run->type regedit)
3. go to HKEY_CURRENT_USER\Software\Google\NavClient\1.1\Options\
4. add a binary value EnableDC with a value of 1 (type 01).
5. go to the Toolbar Options menu item in your toolbar (click the Google
button) and turn on Google Computing!
