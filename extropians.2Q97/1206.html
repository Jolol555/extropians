As the admin for a number of mailing lists, I've found myself the target
of flame mail from a growing number of irate people who have been subscribed
to my lists by a third party, probably as a joke or some sort of revenge. The
main impetus for the last Majordomo upgrade was a new feature that prevents
this sort of harassment. Now when someone subscribes an activation code is
e-mailed to the new address which must be returned before the address is
actually added to the list of subscribers.
