There has been a lot of talk about proper testing of smart drugs on
this list recently.
Is it just talk or are we serious here?

I propose an experiment with ourselves as the subjects:

1) We collectively choose a smart drug to try.

2) Someone slightly skeptical establishes a testing protocol.
( Lee, want to volunteer? )

3) Someone distributes a test ( could be a standard IQ test, or an
attached program that will test hand-eye coordination, or
whatever is appropriate to the drug of choice ) This establishes
a baseline.

4) Have a trusted person blindly distribute it and placebos by snail mail.
( Max, want to volunteer? )

5) Similar tests are re-distributed periodically.

6) At the end of the testing period, test results are correlated
and we can then have real data to argue about.

I hereby volunteer to be one of the guinea pigs, and to possibly
do part 3+5 depending on the drug.

Any takers?

Robert Schrader rms@cts.com

BTW, this could have some publicity value if handled properly...
