Even in quotes, it still cuts off for me. Better to not have spaces in
URLs at all. One can replace unwanted characters in URLs with %XX,
where XX is the hex value for the ASCII character being replaced. (%s,
of course, *must* be replaced, and therefore probably will be in almost
any URL.) I think it's %20 for spaces, but I could be wrong. This
works in the main URL, depending on whether the Web server can translate
%XX (most can; I think it may be in the HTTP spec). Whether it will
work in the #... section is up to the browser.
