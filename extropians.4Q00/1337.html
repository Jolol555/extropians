I'm skeptical about the accuracy of the tests, but I think the categories
can be useful both for self examination and for evaluating others. I'm an
INT*, according to the Kiersey test.

I found an interesting related web site today: http://www.ansir.com. Even
though the test was longer and had more answerable questions than the
Kiersey test, I still think it got the wrong answers. But I really like the
profiles, which are much more detailed than Kiersey. They've also got
something called The Cube Game which was pretty neat. I don't want to give
anything away, but it'd be interesting to see how Extropians fare.
