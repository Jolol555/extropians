Yes, there is an easy way. Get your hands on Ghostscript, which is
Free/Open-source software. Install a postscript printer driver if you
don't have one already (most HP Laserjet printers are postscript
printers) and instruct the driver to print to a file.

Save this file as a .ps file and run ps2pdf on it, which comes with
Ghostscript.
