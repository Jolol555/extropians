Eliezer S. Yudkowsky wrote:
> Genetics is too slow to have any impact, and receives little or no
mention
> among knowledgeable transhumanists.

Transhumanists don't study genetics to understand how it works over the
millennia. We study genetics to learn how the programming works so we can
hack our own code today.

Imagine if we could hack our code to become immortal, immune to disease or
instantly smarter. That would have an impact.
