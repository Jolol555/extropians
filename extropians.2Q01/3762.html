Emlyn wrote:

> Probably no-one here has an opinion about this...
>
> Microsoft Is Set to Be Top Foe of Free Code
> http://www.nytimes.com/2001/05/03/technology/03SOFT.html

I think (1) Microsoft is overreacting to what is basically a
lot of (now mostly dead) hype; (2) Linux is popular because it
fits a particular marketing niche (people who would like to
think they're "power users") and *not* because of anything
inherent in the open source development model, Microsoft
should realise this; (3) Microsoft's stance on the GPL (as un-
American) is most likely sincere, whether I agree with it or
not; and (4) I don't think open source creates better software
(but it doesn't surprise me that a bunch of hackers can make a
relatively stable version of Unix after all these years).
