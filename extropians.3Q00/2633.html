This whole story will, I think, reasonate with the public. It may generate
some flack for Genetic Engineering.

It's very scary. This type of science does carry some real risk. But we are
all dying anyway.

So I say, yes, we may pay some price for GE, but the way things have been
going for humanity so far, we all pay the ultimate price in the end anyway.
So in order to gain this knowledge, we must take risks; if we want to beat
death, risks must be taken.
TANSTAAFL, etc.
