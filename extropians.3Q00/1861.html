Quick question...

I remember a short time ago someone on this list mentioned conferencing
software. The software allowed people to listen in in real-time to a
telephone conference call, but the voice was one-way. The listeners could
reply with computer chat software.

Ideally, I need something that people could use to listen into a meeting,
make comments on in writing, but not have the ability to reply verbally.
I needs to be password accessible. Big pluses if it is open source, needs
only a web browser on the user end, and if it is encrypted.

Any ideas?
