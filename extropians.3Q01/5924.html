> Spare me a clue, Sir? I tried to navigate there, but no dice.

No. The URL was a gateway into the MojoNation anonymous publishing
infrastructure. It is localhost, so it refers to your machine. However,
the reference is correctly resolved, if you install
http://www.mojonation.net/ before. Similiar applies to Freenet.

The successor infrastructure to the web will use steganography to
camouflage as bona fide traffic, and cryptography, to make filtering
impossible. The web is dead, long live the web.

It is still possible to take out individul nodes from where you pulled
slivers of incriminating content by becoming part of the network. I'm sure
laws can be twisted to make such behaviour very illegal, and streamline
persecution, but the world is not universally regulated, and one of course
can use worms to establish a node network on host computers which are run
by innocent users. Here, at least, plausible deniability is truly
plausible.

That's exactly how the shit went down, Your Honour.
