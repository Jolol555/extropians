<Artillo wrote:
Quantum communications?>

The relevant point is being able to signal (at any speed)
with the correlations (entangled particles).

And that is impossible, in principle, holding the locality
condition, which is different from the separability condition
(which is violated by quantum mechanics).

But it does not follow from this that the _string_ of results
does not contain some structure that would allow
superluminal signaling [i.e. 01010101.....].

Perhaps Alice can build some _string_ performing
several measurements, each one on a different pair
of entangled particles, changing the basis every time.
