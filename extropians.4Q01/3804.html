From: Eugene Leitl <Eugene.Leitl@lrz.uni-muenchen.de>

Team ranking up to 50. If you have a box you let running through, consider
downloading and installing the client. The install is a breeze, and the
resource requirements are nonintrusive, even for Windows. Get your clients
here:

        http://folding.stanford.edu/download.html

Only Windows and Linux so far, Mac OS X in beta. Don't let your system
idle senslessly. This is the most worthwhile distributed crunch effort I'm
aware of.

### I had some problems with Folding@home - the machine started warning
about low virtual memory and crashed. I am not authorized to increase the
paging file (now it's at 70MB), so I had to uninstall it. I still have it
running on a couple of other computers (it's the kaufneuro7 team member) but
I wonder if the program could put too much of a load on the LAN. Any advice?
