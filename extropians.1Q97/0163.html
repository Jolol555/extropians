Lyle wrote:
> Apple paid * Four * Hundred * Million * Dollars * for NeXT.
> The more I think about this, the more ridiculous it is.

I can't agree less.

Apple wasted * Five * Hundred * Million * Dollars * developing Copeland, and
if the project hadn't been axed, would still be months away from shipping.

NeXT has the best ( tested, refined, and robust ) object technology
available today, a robust mach-based kernel ( with a much better one in the
labs ), one of the best development environments anywhere ( InterfaceBuilder
& ProjectBuilder ), the best object-rdb mapping framework ( EOF ), a pretty
damn good dynamic web page creation environment ( WebObjects ), and the only
completely transparent OO interprocess communication model anywhere ( PDO ).
There's more stuff that NeXT has in the labs that's even better. Apple got
quite a bargain.
