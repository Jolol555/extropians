A digest of the extropians mailing list is now available. To switch
to the digest version of the list (sent out once a day), send the
following 3 commands to extropians-request@extropy.org:

subscribe extropians-digest
unsubscribe extropians
end

(The "end" command tells the Majordomo list software not to attempt
to process your signature.)

Also, the information file containing the rules of the list has been
updated. To have the file sent to you, send a message containing
"info extropians" to extropians-request@extropy.org.
