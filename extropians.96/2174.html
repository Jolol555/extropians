There will always be list noise. Count on it! How i cope with list noise in
Eudora is I simply read the messages I find most interesting and if i have
more time I'll read the others later. _Everyone_ should practice this. When
I'm done with reading them I dump them to an Archive folder where they sit
and I can go back to them if i wish. It makes it easy to cope with any noise.
