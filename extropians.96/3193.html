If you aren't currently subscribed to the extropians mailing list,
please accept my apologies and notify me (david@kumo.com) immediately.

As most of you have surmised by now, the extropians list broke
earlier this week. Unfortunately the exact nature of the error wasn't
caught in time to restore the subscription list from the nightly backups,
so I've had to reconstruct the list from old backups, recent subscription
requests, and a list of everyone who has posted to the list in the
last 3 months. Consequently there are some old e-mail addresses and
some duplicates that should be unsubscribed. You can send me mail, use the
web-based interface at http://www.lucifer.com/cgi-bin/exi-lists or
send a message to extropians-request@extropy.org with the following
two commands:

unsubscribe extropians <old-email-address>
end

(The "end" command tells the list server to not attempt to interpret
your .sig as commands.)

If anyone has recently sent a "who extropians" command to the list
server and saved the reply, please send it to me as it will be useful
in restoring subscribers who may have been accidently dropped from
the list.

You many want to check out the mailing list archives at
http://www.lucifer.com/exi-lists/extropians/date00.html to see
if you have missed any messages this week.

Rest assured I'm taking steps to ensure this type of error doesn't
happen again and please accept my sincere apologies for any inconvenience.
