Chuck, this is what your message to me looks like:

ERROR: Header line added to ASCII armor: "Hash: SHA1"
ASCII armor corrupted.

Error: Transport armor stripping failed for file pgptemp.$00

Sounds to me like you have Eudora putting ASCII armor on files that are
just signed and not actually encrypted. Some versions of PGP will by
default put all files they touch, encrypted or not, into scrambled text
that looks like it was encrypted, and as a result it can only be read
through pgp. So, turn the ASCII armor feature off.
