    David McFadzean wrote,
    > AOL is blocking all mail sent from extropy.org with this message
    > 550 DIRECT CONNECTION FROM DIAL-UP OR DYNAMIC-IP DENIED
    >
    > Needless to say, the server is not a dial-up or dynamic-ip. The IP is
    > 209.115.169.3 and the reverse DNS resolves to javien2-3.spots.ab.ca. Any
    > suggestions?

    1. The Extropians list is being sent from the server tick.javien.com within
    the javien.com domain.

    2. This server is not an authorized mail exchanger for this domain
    according to the nameserver MX record lookups.
    <http://www.mob.net/~ted/tools/mx.php3?domain=javien.com>.

    3. AOL has started rejecting mail that is sent directly from within a
    domain without going through an authorized mail exchanger for that domain.
    <http://members.aol.com/adamkb/aol/mailfaq/#receiving>.

    Try adding an MX record to the name servers to show tick.javien.com as being
    an authorized mail exchanger for the javien.com domain.
