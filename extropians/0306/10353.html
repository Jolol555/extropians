    Robert J. Bradbury wrote:
    > I think it might work but it is going to need to be defined very very
    > carefully -- does it include "war", does it include "libertarianism",
    > etc.? Then who is going to police things to decide when a discussion
    > thread turns decidedly political -- or can you not mention politics
    > in any way shape or form on the Extro List anymore (its going to be
    > hard for people to self-police themselves on that)?

    I'd suggest the rule that anyone can call for a topic to be moved to
    ExtroPolitics. If there's a brief political digression or a casual
    mention of current events, it might as well stay on Extropians. If it
    turns into a whole long thread, then soon enough someone will call to move
    the topic to ExtroPolitics. Such a call can be overruled by any ExI
    Personnel types if it violates common sense about what is or is not a
    political discussion. Since we're not trying to program an AI, we can
    invoke common sense as a function call.
