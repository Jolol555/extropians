    Why believe the truth? Well, if the questioner thought the truth had no
    intrinsic value then there would be no reason to think an answer would
    have any value and thus no reason to ask the question; if he thought the
    truth did have an intrinsic value then the answer is obvious and there
    would be no reason to ask the question.
    So the real mystery is, why ask the question?
