    Nathanael Allison wrote:

    > I think that Video is a much better medium than Writing.

    You are utterly wrong!

    Video is a good medium for certain types of content. But certainly not
    for all types of content.

    Video simply cannot make information dense enough for certin tasks.

    Try creating a video based math notation, or a video based programming
    language. It simply isn't possible to have enough information "online"
    for the viewer to have an overview of the information.

    So in situations where there is complex interdependencies in the
    information, video plain sucks.

    There is a general rule in programming which says that the number of
    errors grows squared with the amount of the programme that isn't visible
    on the screen at any given time.

    In video only the content our short term memory is "visible" at any
    given time. This makes it nearly impossible to explain complex topics
    with video.
