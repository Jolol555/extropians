    aWARd! I have tried to filter on '[war]', but then many people used the
    'war:' convention instead. So I added that one. But then again there was
    some peple who found it sufficient to just write 'WAR'. Then I changed
    my filter to filter on war, But aparantly there are several english
    words which containt the letters 'war'.

    So here is my main point: meta tags are for computers to filter, not for
    people. So we really should use one, and only one, unique convention!

    I suggest that we use square brackets, as it makes the meta subject
    completely unique, and limited in length. Which a ':' does not.

    [META] ... Please be aWARe to add those meta tags, and to add them
    correctly.
