    At 10:24 AM 5/21/03 -0400, Michael Wiik wrote:

    >Anders Sandberg makes a handy chart:
    >
    >>Tech Social Economic
    >> - - - Anti tech authoritarians (reactionaries)
    >> - - + Anti tech conservatives (Kass, Fukyama)
    >> - + - Anti tech liberals (Rifkin)
    >> - + + Anti tech libertarians (small is good?)
    >> + - - Pro tech authoritarians (high tech fascism,prometheans)
    >> + - + Pro tech conservatives (good for business)
    >> + + - Pro tech liberals (left transhumanism?)
    >> + + + Pro tech libertarian ("classical" extropianism)

    Good but incorrect. As much as I like Ander's work on this example, after
    consulting with Max, we concur that Pro tech libertarian is not "Classical"
    extropianism. It is "Early" extropianism. Very early, in fact.

    Let's not confuse classical with early and certainly not label any pro tech
    extropian under any one political party belief system. This in itself is
    not extropian.
