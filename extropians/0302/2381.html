    Eliezer S. Yudkowsky asked, "Has anyone else begun to get the feeling
    that, wherever you go, you are there to represent the Net groupmind?"

    I had a similar experience. I used Netscape 4.7 for archiving my email
    and recently switched to 7.01. I also sometimes send messages from
    Netscape when I am not using MUTT. Anyway, I did a MS upgrade and
    Netscape could no longer open up a new "compose" window. After deleting
    and reinstalling the code and considering Netscapes "paid" support
    services, I searched on the error message in Google.

    I found the bug log for the code and read about many peoples experience
    related to several different bugs with the same error. The net result
    was that many people had added a new account and that fixed things. I
    added a new (bogus) email account to Netscape and that fixed it..

    So yes, as Eliezer spoke about, I was able read and learn about a
    problem outside my own "ken." For me it was magical in the sense that I
    would never have guessed the correct solution.
