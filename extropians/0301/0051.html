Happy New Year fellow extropes!

I heartily recommend this cute little time waster:
http://www.twilightgames.com/download/dargon10.exe (approx 2.7 MB)

Aargon Deluxe is a stimulating logic game in which you manipulate one or
more laser beams using mirrors, refractors, splitters, prisms, filters etc.
to target coins of various colors. The object of the game is to spin the
coins using only the apparatus available for each level. The game is truly
engrossing from the very first levels - you won't stop until you've played
it to the end! This free demo version (which doesn't expire unlike most
demos) allows you play several of the early levels and the tutorial gets you
up and running in minutes.

Enjoy!
Russell Evermore
