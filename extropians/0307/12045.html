    What seems to be overlooked is that a properly constructed moral code
    will not permit either the extermination of the species nor endorse the
    murder or other abuse of individuals to serve the needs of the species.
    In the event one finds an unresolveable conflict, then it is one's code
    that is incomplete or self-contradictory, not reality.

    One reason our code of law here uses the common law system is that it
    assumes that, barring a limited set of general strictures that cannot
    be crossed (i.e. individual liberties, etc), everything else is
    negotiable or resolvable in an evolutionary process of increasing refinement.
