    Hello,

    As one of the several list moderators, I noticed this morning several
    replies where messages were quoted largely intact with brief responses.
    This proves to be burdensome because one actually has to spend time
    scanning the message to find the actual response and then puzzle over
    perhaps what points in the original message/conversation were being
    commented on.

    We all have limited bandwidth. I am certain that there are some list
    members that tend to scan based on message size (small messages get
    read, large messages don't). It helps the list <significantly> if,
    when one writes a reply, one takes the time to trim a response to only
    include the most significant part(s) of the message being responded to.
    We have the list archives as well as the Javien forum -- it is very
    easy for people to go to these sources to determine the context of
    a response if it is unclear from any quoted parts.

    Thank you,
    Robert
