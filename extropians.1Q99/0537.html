Eliezer S. Yudkowsky, <sentience@pobox.com>, writes:
> I think I can now extract the basic point of contention:

> Subjectivist: If there were an objective morality, I would have found it

I don't agree. The real question is whether there is any meaning to the term "objective morality". I don't know a definition which avoids circularity.

Objective morality: a system for telling which actions are moral. Moral: the quality of actions in keeping with objective morality.
