As I've just found out, using a Bcc: to the Extropians list will cause the forwarded message to go into my main mailbox; right past Netscape's mail filter, which can only check To: and not Reply-To:, like these guys never heard of using a filter on a mailing list.

I Bcc'd because I was also sending a copy to original@netfunny.com and didn't want anyone's reply getting crosswired; so I Bcc'd both of them. RHF's got an autoresponder; didn't want a repeat of the Great Gingrich Flood.

Sorry if I contaminated anyone's mailbox; put it down to lessons learned.

Bcc stands for Blind carbon copy.
