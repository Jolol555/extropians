Oops. I'm sending more duplicate postings. In addition to extropioans
at extropy.org and extropy.com, I am now seeing extropians at
maxwell.kumo.com and some gnu.org site.

Are poeple setting up mirrors for the extropians list? Or, are people
using different addresses to send to the extropians list? These
multiple addresses are confusing various software. Normally, a list has
one name, and everybody sends to the one name. A reply to the list
should send a single reply to the single list. When all these different
names are used, it makes it difficult for the software to figure out
which one to reply to. In some cases, the multiple names for the list
each get their own reply.

Is there some way to get MajorDomo to use "To: extropians@extropy.com"
instead of using whatever the poster decided to use? Setting the "To:"
and "Reply-To:" fields to two different addresses that actually go to
the same list, is just asking for duplicates.
