A very good question.

I think clarity is very linked with being intelligent - patterns,
connections and chains of reasoning are easily held in the mind,
without getting lost in the jumble. This suggests to me that working
memory is one of the big bottlenecks of high intelligence, another one
is ability to concentrate. Too bad that I'm more of a long-term memory
guy than a working memory guy, I cannot estimate how easy/hard it
might be to improve the working memory system.
