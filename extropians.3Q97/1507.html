NanoCAD in Java is now basically functional, and should run on any
Java-capable browser (although I had trouble trying to use it with a
friend's ATTnet account, using Internet Explorer). Chris Fry has come
up with a lot of really useful suggestions for making the code cleaner
and the user interface more straightforward. Anybode interesting in
taking it for a spin should visit

http://world.std.com/~wware/ncad_java/nc.html

I believe (but at this point I'm not yet sure) that you can also get
to it with the old address:

http://world.std.com/~wware/ncad.html

still a pretty inexperienced Java programmer; I think I've gotten
everything right but I'd be glad for any feedback.
